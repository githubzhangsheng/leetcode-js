### 二叉树

### 二叉树的三种遍历方式

- 前序遍历（根、左、右）
- 中序遍历（左、根、右）
- 后序遍历（左、右、根）

常见的方法使用递归的方式来求解，代码简单而且易于理解。
当掌握了递归的方式来遍历二叉树后，一些面试官会要求给出迭代遍历二叉树的算法。
在二叉树的迭代遍历算法中，一般可以借助栈来记住遍历过的节点，其中的难点是二叉树后序遍历的时候一些节点会出现二次遍历的情况，此时需要判断该节点是否已经被遍历过。
