## 数据结构

首先需要了解常用的数据结构，这里从二叉树开始，二叉树的数据结构比一般的结构要稍微复杂些，也是大厂最喜欢考的数据结构的类型
先学会二叉树是绝对不亏的。

### 二叉树

二叉树的三种遍历方式：

- 前序遍历（根、左、右）
- 中序遍历（左、根、右）
- 后序遍历（左、右、根）

常见的方法使用递归的方式来求解，代码简单而且易于理解。
当掌握了递归的方式来遍历二叉树后，一些面试官会要求给出迭代遍历二叉树的算法。
